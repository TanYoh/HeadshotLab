# HeadshotLab

一个轻量的网页 FPS 枪法训练小游戏。

`HeadshotLab` 的核心玩法很直接：在限定时间内，快速移动并点击随机出现的目标，尽可能拿到更高分数和更高命中率。它适合拿来做短时反应训练，也适合作为一个打开就能玩的浏览器小项目。

## 游戏简介

- 支持 `60s` / `120s` 两种训练时长
- 支持慢 / 中 / 快三档出点节奏
- 提供实时数据面板：剩余时间、出现目标数、命中率、当前分数
- 训练结束后会自动保存成绩
- 内置最近记录和当前配置下的最佳成绩展示
- 首次开始前会弹出计分说明，帮助玩家理解得分规则

## 玩法说明

1. 选择训练时长、出点间隔和界面大小。
2. 点击“开始训练”。
3. 在训练区域内快速点击随机出现的目标。
4. 越接近靶心，得分越高。
5. 时间结束后，系统会展示并保存本次成绩。

## 本地运行

这个项目不依赖构建工具，直接打开即可运行。

```bash
# 方式一：直接双击 index.html
# 方式二：在浏览器中打开 index.html
```

项目主要文件：

- `index.html`：页面结构
- `styles.css`：界面样式
- `script.js`：游戏逻辑、计分、记录存储

## 成绩记录

成绩会保存在浏览器本地。项目会优先使用更现代的本地文件能力进行存储，不可用时回退到 `localStorage`，所以刷新页面后记录通常会保留。
